# Reconnaissance scripts

## This folder contains reconnaissance custom scripts that either automate or simply gathering information on target systems.

### A few mitigation solutions
1. Implement firewalls to filter incoming traffic.
2. Use IDS/IPS to monitor possible attacks and stop them in the early stages.
3. Block access to well-known port numbers or change what port a specific service uses to some lesser-known port numbers such as those from the ephemeral range (49152 through 65535).

This folder is not in it's final form, so there will be a lot of changes happening regarding both documentation and scripts.
