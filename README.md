# Cybersecurity projects

## About the repository
This repository contains cybersecurity related projects which are meant only for educational purposes.
The projects presented here have been implemented in controlled environments by either using GNS3 or virtual environments such as VMware or VirtualBox.

> [!CAUTION]
> The contents of this repository are not supposed to be used to conduct any real cyber attacks/crimes.
> I discourage the use with malicious intent of the available scripts inside this repository.
